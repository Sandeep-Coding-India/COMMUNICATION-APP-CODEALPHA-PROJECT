# Video Chat Application 
 
A complete real-time video communication web application with WebRTC, Socket.io, MongoDB, and JWT authentication. 
 
## Installation 
 
1. Make sure MongoDB is installed and running 
2. Run: npm install 
3. Run: npm start 
4. Open browser: http://localhost:3000 
 
## Features 
 
- User Registration & Login 
- Multi-user Video Calling 
- Real-time Chat 
- Screen Sharing 
- File Sharing 
- Collaborative Whiteboard 
- User Notifications 
 
## Usage 
 
1. Register a new account 
2. Login with your credentials 
3. Create a new room or join existing room 
4. Share room ID with others 
 
For detailed documentation, see the complete README in the project.
