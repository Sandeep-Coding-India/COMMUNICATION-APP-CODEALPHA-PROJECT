HOW TO RUN THE APPLICATION: 
================================================ 
 
STEP 1: START MONGODB 
  Make sure MongoDB is running on your system. 
  If not installed, download from: https://www.mongodb.com/try/download/community 
 
  To start MongoDB: 
  - Windows Service: net start MongoDB 
  - OR run manually: mongod 
 
STEP 2: START THE SERVER 
  Open terminal in: E:\CODE\COMMU\video-chat-app 
  Run: npm start 
  OR: node server/server.js 
 
STEP 3: OPEN IN BROWSER 
  Go to: http://localhost:3000

FIRST TIME USAGE: 
================================================ 
 
1. Click "Register"

TROUBLESHOOTING: 
================================================ 
 
Q: MongoDB connection error? 
A: Make sure MongoDB is running on port 27017 
 
Q: Camera/mic not working? 
A: Allow browser permissions for camera and microphone 
 
Q: Port 3000 in use? 
A: Change PORT in .env file
